# Training Custom Data On Mobilenet V2

## **Technique Used to bring all images to 224 * 224**
1. Pad (500 x 500)
2. Resize (250 x 250)
3. CenterCrop (224 x 224)

**Maximum Train Accuracy** - 93.65%

**Maximum Test Accuracy** - 85.85%
